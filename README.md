# German Credit Risk Predictor

This project is a complete **machine learning solution** to predict credit risk for German customers. Using the German Credit dataset, it identifies whether a client is **high-risk or low-risk**, helping banks and financial institutions make data-driven decisions. The project covers the full ML pipeline, from **data preprocessing** and **feature engineering** to **model training, evaluation, and deployment**.

The interactive **Streamlit dashboard** allows users to test the model with real-time inputs and visualize credit risk patterns based on client attributes such as age, sex, housing, and savings accounts.

---

## Features

- Predicts whether a client is **high-risk or low-risk**  
- Provides visual insights into patterns of credit risk by **age, sex, housing, and savings accounts**  
- Saves **label encoders** for reproducibility and consistent predictions  
- Interactive **Streamlit app** for real-time testing and demonstration  
- Supports multiple machine learning models: **Decision Tree, Random Forest, Extra Trees, and XGBoost**  

---

## Tech Stack

- **Python** – programming language for data analysis and modeling  
- **Pandas & NumPy** – for data manipulation and numerical operations  
- **Scikit-learn & XGBoost** – for machine learning model development  
- **Matplotlib & Seaborn** – for data visualization  
- **Streamlit** – for building an interactive web dashboard
