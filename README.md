# German Credit Risk Predictor

Predicting credit risk using machine learning techniques on the German Credit dataset. This end-to-end ML project handles data cleaning, preprocessing, feature encoding, model training (Decision Tree, Random Forest, Extra Trees, XGBoost), and evaluation using accuracy, confusion matrix, and classification reports. It also includes an interactive **Streamlit** dashboard for real-time predictions.

**Features:**  
- Predicts whether a client is high-risk or low-risk  
- Visualizes credit risk patterns by age, sex, housing, and savings accounts  
- Saves label encoders for reproducibility  
- Interactive Streamlit app for real-time testing  

**Tech Stack:** Python, Pandas, NumPy, Scikit-learn, XGBoost, Matplotlib, Seaborn, Streamlit
